Details on supervision

The type of supervision will depend on whether there are sufficiently many students working on the same project as you. Overall, you should expect less supervision with projects on which very few students are working. So if you select such a project, you should feel comfortable working independently.

  • Sufficient students on same project:

    • During block 1: We will meet in groups and you (the students) will present the background literature and theory, and we will have time for questions, both general questions as well as questions specific to what you are reading.

    • During block 2: We will meet individually. At these meetings you are expected to shortly present your progress and have a set of prepared questions you would like to discuss.

  • Insufficient students on same project:

    • During block 1: Short individual meetings. The goal of these meetings is to ensure that you spend time on relevant things, so you should prepare a detailed plan of what you plan to work on for each of these meetings.

    • During block 2: We will meet individually. At these meetings you are expected to be shortly present your progress and have a set of prepared questions you would like to discuss.

Individual meetings can be onsite or on zoom.

Prerequisites

The formal prerequisites are the courses Statistical Methods and Mathematical Statistics (or equivalent), but you are also expected to be interested in the following:

  • carry out data analysis and model validation on real data
  • implement models and/or data analyses (e.g. by writing R or Python scripts)
  • learn to use new software packages and functions
  • find relevant literature
  • independently read up on the background theory of the project
  • write a project that reflects theory as well as applications

Project suggestions

Project 1: Biomass prediction - Comparing confidence intervals

Quantifying the biomass due to trees is important to assess the stored carbon in a specific region. However, estimating the entire biomass in a region is a non-trivial task that often involves counting and measuring tree sizes. Recent advances in aerial and satellite remote sensing have provided high-resolution images of regions that together with deep-learning image analysis can be used to count each tree and provide its crown area and sometimes even height. Given individual tree samples for which the biomass as well as crown area and height has been recorded, one can learn a prediction model to predict biomass from crown area and height. This model can then be applied to the remote sensing data to estimate the biomass in large areas.

The prediction models that have been used previously are simple power-law models that are generally fit using a log-log transformation in order to simplify the estimation. A further approach is to use non-linear least squares to directly fit the power-law models, however, this is often rejected by practioners as non-linear least squares does not come with the standard model-based prediction intervals. In this project, you should investigate, based on the tree biomass data, how the coverage of different types of non-parametric prediction intervals compare to the currently used log-log transform based intervals.

The final report should include an overview of the data as well as a short introduction to any of the methods that were used in the analysis. The main part of the project should then outline the discoveries including a clear description of how the different prediction models were compared. The literature list below provides tutorials on two approaches to construct non-parametric prediction intervals: conformal prediction and bootstrap.

Data

For this project you will analyze the data set used in Allometric equations to estimate the dry mass of Sahel woody plants mapped with very-high resolution satellite imagery. It consists of samples of tree foilage, wood and root masses and corresponding crown area and height.

Literature

  • A Gentle Introduction to Conformal Prediction and Distribution-Free Uncertainty Quantification
  • The bootstrap: a tutorial

Project 2: Human gut microbiome - Distribution generalization

Microorganisms, such as bacteria, fungi and viruses interact in various ways with their surroundings. The human body is estimated to be a habitat for more than 10’000 different microbial species and they have been associated with various health outcomes such as cardiovascular disease, metabolic diseases, and obesity. Thanks to recent advances in gene sequencing technology the field of microbiome research, which measures the microorganisms based on gene sequencing, has made significant progress on understanding the links between the microbiome and its host.

One application of microbiome measurements is to use them as a diagnostic tool. For this one learns a predictive model that uses the microbiome to predict a specific phenotype that might be hard to measure in practice. An important problem with predictive models is that they may not generalize well to new observations with different characteristics than in the training sample. In this project, you will investigate how to construct predictive models based on microbiome predictors that are robust to distributional shifts. You will work with the publicily available ‘American Gut Project’ data, which contains microbiome measurements and corresponding metadata (e.g., age, sex, BMI). A potential avenue to investigate might be to consider different types of variable selection/screening procedures and analyze how they affect generalization performance.

The final report should include an overview of the data as well as a short introduction to any of the methods that were used in the analysis. The main part of the project should then outline the discoveries including a clear description of how the different prediction models were compared. The literature list below provides several tools and methods that can be used in the analysis.

Data

For this project you will use data from the American Gut Project http://americangut.org. This is an initiative that enrolls regular people around the world (mainly USA and UK) and collects data from each participant (consisting of a gut microbiome sample and a questionnaire).

Literature

  • American Gut: an Open Platform for Citizen Science Microbiome Research. Paper describing the American Gut Project.

  • https://github.com/knightlab-analyses/american-gut-analyses Github repository for American Gut Project, providing details on downloading full data (not required for this project but may be useful to get more information on it).

Some tools that might help in the analysis:

  • Supervised Learning and Model Analysis with Compositional Data. A method for a regression-based analysis that adjust for the compositional nature of the sequencing data. Method is available as a python package.

  • Regression models for compositional data: General log-contrast formulations, proximal optimization, and microbiome data applications. Extension of the log-contrast model to include sparsity penalty. Method is available as a python package.
