Jupyter Notebooks (in R)

Jonas Peters and I have created some jupyter notebooks. They give a self-contained overview of a few selected topics related to causality and include several coding exercises. Currently, we have notebooks for the topics

  • Computing Interventions
  • Instrumental Variables
  • Restricted Structural Causal Models
  • Invariant Prediction

They can be downloaded here. The package contains a “readme” with some instructions for installing jupyter notebook (we apologize if the instructions are out-of-date). Most importantly, you may want to run “setupNotebook.ipynb” first to make sure that everything is working.
